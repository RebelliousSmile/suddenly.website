# Déploiement Alwaysdata

**Difficulté** : Facile | **Coût** : ~10 €/mois

Alwaysdata est une plateforme d'hébergement managé qui simplifie le déploiement. Elle gère le serveur web, la base de données PostgreSQL et les certificats SSL automatiquement.

> Alwaysdata ne propose pas Redis — Suddenly bascule automatiquement sur le cache base de données si `REDIS_URL` est absent. Celery tourne en mode synchrone (`CELERY_TASK_ALWAYS_EAGER=True`).

---

## 1. Créer les ressources dans le panel

1. **Base de données PostgreSQL** — noter l'hôte, le port, le nom, l'utilisateur et le mot de passe
2. **Site Python WSGI** — créer un site (voir configuration à l'étape 6)

---

## 2. Cloner le dépôt en SSH

```bash
ssh user@ssh-user.alwaysdata.net
cd /home/<user>/www/
git clone https://github.com/RebelliousSmile/suddenly.git soudainement
```

---

## 3. Installer les dépendances

```bash
cd /home/<user>/www/soudainement
python3.12 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install -e ".[federation]"
```

---

## 4. Générer les clés ActivityPub

```bash
mkdir -p /home/<user>/www/soudainement/keys
cd /home/<user>/www/soudainement/keys
openssl genrsa -out private.pem 2048
openssl rsa -in private.pem -pubout -out public.pem
```

---

## 5. Initialiser la base de données

Les variables d'environnement du panel ne sont pas disponibles en SSH. Il faut les exporter manuellement :

```bash
export DJANGO_SETTINGS_MODULE=config.settings.production
export SECRET_KEY="votre-cle-secrete"
export DOMAIN=soudainement.fr
export ALLOWED_HOSTS=soudainement.fr,www.soudainement.fr
export DATABASE_URL="postgresql://user:password_encodé@host:5432/dbname"
```

> **Important** : encoder les caractères spéciaux du mot de passe :
> ```bash
> python3 -c "from urllib.parse import quote_plus; print(quote_plus('mot_de_passe'))"
> ```

```bash
cd /home/<user>/www/soudainement
source venv/bin/activate
python manage.py migrate
python manage.py createcachetable
python manage.py collectstatic --noinput
python manage.py createsuperuser
```

---

## 6. Configurer le site dans le panel

| Champ | Valeur |
|-------|--------|
| Chemin de l'application | `www/soudainement/suddenly/wsgi.py` |
| Répertoire de travail | `www/soudainement/` |
| Version Python | `3.12` |
| Répertoire du virtualenv | `www/soudainement/venv` |
| Chemins statiques | `/static/ staticfiles/` et `/media/ media/` |

> **Pièges** :
> - Le fichier WSGI est `suddenly/wsgi.py`, **pas** `config/wsgi.py`
> - Les chemins statiques sont **relatifs au répertoire de travail** (ne pas mettre le chemin absolu)
> - `DATABASE_URL` : encoder les caractères spéciaux du mot de passe

### Variables d'environnement (format `FOO=bar`, sans `export`)

```
DJANGO_SETTINGS_MODULE=config.settings.production
SECRET_KEY=votre-cle-generee
DOMAIN=soudainement.fr
ALLOWED_HOSTS=soudainement.fr,www.soudainement.fr
DATABASE_URL=postgresql://user:password_encodé@host:5432/dbname
```

---

## 7. SSL

Activer Let's Encrypt via le panel Alwaysdata pour le domaine.

---

## 8. Tâches planifiées

Dans le panel → **Tâches planifiées**. Renseigner les variables d'environnement dans chaque tâche (même format que le site).

**Nettoyage des sessions** (quotidien à 3h) :
```
Fréquence : 0 3 * * *
Commande  : /home/<user>/www/soudainement/venv/bin/python /home/<user>/www/soudainement/manage.py clearsessions
```

**Nettoyage des citations éphémères** (toutes les heures) :
```
Fréquence : 0 * * * *
Commande  : /home/<user>/www/soudainement/venv/bin/python /home/<user>/www/soudainement/manage.py shell -c "from suddenly.activitypub.tasks import cleanup_expired_quotes; cleanup_expired_quotes()"
```

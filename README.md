# suddenly.website

Site de présentation de [Suddenly](https://github.com/RebelliousSmile/suddenly), réseau fédéré de fiction partagée pour les joueurs de jeu de rôle.

## Stack

- [Nuxt 4](https://nuxt.com) + Vue 3
- [UnoCSS](https://unocss.dev)
- [@nuxtjs/i18n](https://i18n.nuxtjs.org) — français / english
- Polices : Inter, Crimson Text, Playfair Display

## Développement

```bash
pnpm install
pnpm dev
```
